[ 8ball / bola8 ]

Este comando generara una respuesta aleatoria, con respecto a tu pregunta.
EJEMPLO:   ro!8ball ¿soy real?
[ hello / hola ] Este comando hara que te salude, usando el sistema de menciones, y sepas, que te estoy respondiendo.
EJEMPLO:   ro!hello
[ neko / 0w0 ] Este comando hara que envie una imagen de un personaje anime "Neko" aleatorio.
EJEMPLO:   ro!neko
[ ping ] Si quieres jugar pong con un GIF, usa este comando.
EJEMPLO:   ro!ping
[ prob / pbd ] Si quieres jugar pong con un GIF, usa este comando.
EJEMPLO:   ro!prob ¿pasare la materia :v?
[ say / decir ] Si quieres que diga algo, usa este comando, para decir lo que quieres que escribas.
EJEMPLO:   ro!say soy programador
